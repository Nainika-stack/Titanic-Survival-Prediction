# Titanic Survival Prediction

This project applies Machine Learning techniques to predict passenger survival in the Titanic disaster dataset from Kaggle.

## Project Overview

The objective is to build a classification model that predicts whether a passenger survived based on demographic and travel-related features such as age, gender, passenger class, fare, and family size.

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn

## Workflow

1. Data Loading and Exploration
2. Data Cleaning and Missing Value Handling
3. Exploratory Data Analysis (EDA)
4. Feature Engineering
5. Data Preprocessing
6. Model Training using Random Forest Classifier
7. Prediction Generation
8. Kaggle Submission

## Key Features

* Missing value treatment
* Categorical variable encoding
* Feature selection
* Random Forest classification model
* Submission file generation for Kaggle

## Results

The model was trained on the Titanic training dataset and used to generate predictions for the Kaggle Titanic Machine Learning Competition.

## Learning Outcomes

* Understanding of supervised machine learning
* Data preprocessing and feature engineering
* Classification model development
* Model evaluation and prediction workflow
* End-to-end machine learning project implementation

Dataset Source: Kaggle Titanic - Machine Learning from Disaster
